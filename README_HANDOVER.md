# Prism Handover Guide

This file is a quick map for handing the project folder over to a supervisor.
The main working track is the IEQ paper, not the full historical contents of
the repository.

## Most Important Files

1. Modeling dataset
   - `02_Datasets/model_ready/ieq_model_dataset.xlsx`

2. Main notebooks
   - `03_Code/ieq_paper/01_notebook/02_ieq_all_models_hyperparameter_tuning.ipynb`
   - `03_Code/ieq_paper/01_notebook/03_ieq_tuned_all_models_comparison.ipynb`
   - `03_Code/ieq_paper/01_notebook/04_ieq_extra_trees_fine_tuning.ipynb`
   - `03_Code/ieq_paper/01_notebook/05_ieq_final_extra_trees_model.ipynb`

3. Shared figure style
   - `03_Code/ieq_paper/01_notebook/paper_style.py`
   - Notebooks `02` through `05` import plotting configuration, colors, and helper functions from this file.
   - To change the appearance of paper figures, edit this file first. The change will propagate across figures generated by the notebooks.

4. Main analysis outputs
   - `03_Code/ieq_paper/02_outputs/tables/`
   - `03_Code/ieq_paper/02_outputs/figures/`
   - `03_Code/ieq_paper/02_outputs/all_models_comparison/`
   - `03_Code/ieq_paper/02_outputs/final_extra_trees_model/`

5. Figures used by the paper
   - `04_Figures/`

6. LaTeX draft
   - `06_Paper/IEQ.tex`
   - `06_Paper/IEQ.pdf`
   - `06_Paper/references.bib`
