# Related papers

Literature is organized only in numbered folders.

- `01_Core_Overall_IEQ`: overall IEQ models, acceptance, and satisfaction papers.
- `02_SensorBased_IEQ`: sensor-based IEQ monitoring and prediction.
- `03_Classroom_IEQ`: classroom, school, and education-building IEQ papers.
- `04_Methods_Standards`: methodological papers and standards used by the analysis.
- `05_Reviews`: review papers.

The older folders `A Dataset related`, `B Overall IEQ`, and `Both` were merged
into this structure and removed.
