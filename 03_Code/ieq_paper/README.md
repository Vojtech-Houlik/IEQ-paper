# IEQ Paper

This folder contains the working track for the overall IEQ paper.

## Main Handover Files

- Main model dataset: `../../02_Datasets/model_ready/ieq_model_dataset.xlsx`
- All-model hyperparameter tuning notebook: `01_notebook/02_ieq_all_models_hyperparameter_tuning.ipynb`
- Tuned all-model comparison notebook: `01_notebook/03_ieq_tuned_all_models_comparison.ipynb`
- Extra Trees fine-tuning audit notebook: `01_notebook/04_ieq_extra_trees_fine_tuning.ipynb`
- Final Extra Trees model notebook: `01_notebook/05_ieq_final_extra_trees_model.ipynb`
- Paper tables and intermediate figures: `02_outputs/figures/`, `02_outputs/tables/`
- Tuned all-model comparison outputs: `02_outputs/all_models_comparison/`
- Final Extra Trees model outputs: `02_outputs/final_extra_trees_model/`
- LaTeX figure copies: `../../04_Figures/`
- LaTeX draft: `../../06_Paper/IEQ.tex`
- Compiled PDF: `../../06_Paper/IEQ.pdf`

## Structure

- `00_progress`: progress log and PDF exports
- `01_notebook`: main notebooks only
- `02_outputs/figures`: generated intermediate figures
- `02_outputs/tables`: generated result tables
- `02_outputs/all_models_comparison`: outputs generated by the tuned all-model comparison notebook
- `02_outputs/final_extra_trees_model`: final model artifacts, validation tables, and model-inspection figures
- `02_outputs/supplementary_experiments`: supporting experiments and checks

The root-level `../../04_Figures` folder is reserved for manuscript-ready IEQ
figures that are copied into the LaTeX article. Working outputs stay in
`02_outputs`.

## Notes

The notebook sequence starts at `02` because dataset preparation is step `01`
and is handled by scripts in `../_project_tools/`, not by a supervisor-facing
notebook. The current handover flow is:

1. tune hyperparameters for all candidate models,
2. compare all tuned model families,
3. audit Extra Trees fine-tuning options,
4. train and save the final Extra Trees model.
