# Thermal Comfort Paper

This folder contains the complete working track for the thermal comfort paper.

## Structure

- `00_progress`: progress log and PDF exports
- `01_data_raw`: raw source data
- `02_data_clean`: cleaning notes and data-definition notes; clean/model-ready Excel datasets live in `../../02_Datasets`
- `03_notebook`: one main notebook for the paper workflow
- `04_outputs/figures`: generated figures
- `04_outputs/tables`: generated tables
- `05_literature`: citation notes and literature matrix
- `06_manuscript`: manuscript outline and draft text
- `07_transfer_learning`: external dataset and transfer experiments

## Main Notebook

Use `03_notebook/thermal_comfort_workflow.ipynb` as the single reproducible workflow for data cleaning, modeling, figures, and tables.
